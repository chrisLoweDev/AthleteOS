_This file is generated by `/setup` and updated by `/fetch-activities`. Do not edit routine structure manually — use `/edit-plan` or re-run `/setup` to regenerate._

> **Important:** Exercise names in this library must exactly match entries in `data/hevy-exercises.json`. If you add a new exercise, verify the name against the cache (or run `/sync-hevy-exercises` then search the JSON) before adding it here — a mismatch will cause `/push-workouts` to fail with exit code 3.

> **Hevy push note:** When generating individual workout files from this library, `push_hevy.py` reads three sections: `## Main Lifts`, `## Accessory Work`, and `## Core`. The `## Warm Up` and `## Mobility Close` sections are **not** pushed — they are for athlete reference only. Always preserve these section headers exactly when generating workout files.
